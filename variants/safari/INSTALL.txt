LexPack 0.8.4-safari — прототип для Safari на macOS

Safari устанавливает расширение вместе с приложением LexPack Safari. Этот
каталог Web Extension не предназначен для рассылки как обычный ZIP Chrome.

Для локальной разработки выполните npm run build:safari-app, откройте
safari/LexPack Safari/LexPack Safari.xcodeproj и запустите схему LexPack Safari
для My Mac. Затем включите LexPack Safari в Safari → Настройки → Расширения и
разрешите доступ к сайтам consultant.ru. В Safari → Настройки → Сайты →
Загрузки установите для online.consultant.ru значение «Разрешить»; общую
настройку для остальных сайтов можно оставить «Спросить».

Приложение запрашивает доступ только к папке «Загрузки». Готовые документы и
JSON-отчёт сохраняются в подпапку LexPack (либо в указанную в popup подпапку).

Перед рабочей рассылкой приложение необходимо подписать и нотарифицировать.
