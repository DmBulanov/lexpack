Cons Download 0.7.1 — установка для пользователя

1. Распакуйте cons-download.zip в постоянную папку. Не удаляйте её после установки.
2. Откройте в Google Chrome адрес chrome://extensions.
3. Включите «Режим разработчика».
4. Нажмите «Загрузить распакованное расширение».
5. Выберите распакованную папку, в которой рядом находятся manifest.json и этот файл.
6. Закрепите значок Cons Download и войдите в online.consultant.ru.

Для обновления замените содержимое этой же папки файлами новой версии и нажмите
«Обновить» на карточке Cons Download в chrome://extensions.

Коллеге для установки передают только исходный файл cons-download.zip. Не передавайте
профиль Chrome, логин, пароль, cookies, скачанные акты или JSON-отчёты вместе с
установочным архивом.

JSON-отчёт создаётся после конкретной массовой выгрузки. Это контрольная ведомость
для сверки состава и статусов; для работы расширения она не нужна.
